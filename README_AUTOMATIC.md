## ğŸ› Troubleshooting

### JARVIS not hearing wake word
- Check microphone permissions
- Speak clearly: "Hello JARVIS"
- Try: "Hey JARVIS" or "Hi JARVIS"

- Speak more clearly
- Use longer sentences
- Reduce background noise

- Press **ESC** key
